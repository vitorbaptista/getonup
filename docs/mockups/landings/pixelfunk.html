HOW IT WORKS

Three stages.
One groove.

From a file on your machine to a link you can text a friend — no dashboard, no waiting, no build pipeline. Clear each stage and you're live.

STAGE 1

Your agent builds it

A coding agent spits out an HTML file, a React or Vue component, a whole static folder. The artifact is ready to roll.

STAGE 2

Run the command

One line: getonup deploy <file>. It uploads, auto-wraps, and provisions in seconds flat.

STAGE 3

Share the live link

You get back a real, public URL. Send it, embed it, demo it. It just works — and idles right back to zero.

One Worker + R2

The whole backend is a single Cloudflare Worker over R2 storage. Scales to zero, costs about $0 when nobody's looking.

ITEM 02

Agent-native

Built for coding agents to call directly. They publish their own work and hand you a link — no human in the loop.

Auto-wrap, no build

Bare React, Vue, or TypeScript components get wrapped into runnable pages. No bundler, no config, no build step.
