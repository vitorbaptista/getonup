Your AI artifact, live in seconds.

Conjure turns any AI-generated web artifact — an HTML file, a React or Vue component, a built static folder — into a live, shareable URL with one command. One Worker, one bucket, no database. The self-hostable take on one-click publishing.

Three quiet steps

From a file to a link, without the ceremony.

No dashboards to click through, no project to configure. Your agent writes a file, runs a command, and shares the URL.

1

Build an artifact

Your coding agent writes a single .html, .tsx, or .vue — or a built static folder. Whatever it just made.

2

Run one command

conjure deploy auto-wraps the component into a self-contained page and uploads it to your bucket.

3

Share the link

You get a clean, public URL. No viewer login, no always-on server — it scales all the way to zero.

Why it feels different

Small, sharp, and entirely yours.

Conjure is deliberately tiny — one Worker, one bucket, and a CLI any agent can call. Nothing to babysit, nothing to outgrow.

One Worker + R2

No database, no always-on server, nothing to keep warm. A single Cloudflare Worker fronts an R2 bucket and sits at roughly $0 while idle.

Agent-native

Drop one snippet into AGENTS.md and any coding agent can publish on its own — no human in the deploy loop.

Auto-wrap

Hand it one .tsx or .vue and it becomes a runnable page — React 18 + Babel + Tailwind, or Vue 3, all self-contained.

Yours to host

Your Cloudflare account, your token, your data. Free and MIT-licensed, so you can read it, fork it, and run it forever.
