How it works

From artifact to URL in three steps

No database. No always-on server. No viewer login. Just one command between your agent and a shareable link.

1

Build an artifact

Your agent writes a .html, .tsx, or .vue file — or a built static folder. Anything that renders in a browser.

2

Run one command

conjure deploy auto-wraps the component into a self-contained page and uploads it to your own backend.

3

Share the link

You get a live URL instantly. No viewer login, no cold-start wait, and it scales to zero when idle.

Why Conjure
