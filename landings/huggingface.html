How it works

From component to public link in three steps.

No build pipeline to configure. Point Conjure at a file and it does the rest.

1

Build an artifact

Your agent writes a .html, .tsx, or .vue — or a built static folder. Whatever it just generated, you're ready.

counter.tsx · dashboard.vue · index.html
2

Run one command

One CLI call auto-wraps single components into self-contained pages and uploads them straight to your bucket.

$ conjure deploy counter.tsx
3

Share the link

Get a clean, shareable URL instantly. No viewer login, no signup wall, and it scales to zero when idle.

✓ live → /s/iwcmg3dt
